哈希表（Hash table）

  • 散列表（hash table）也叫哈希表，根据关键码值（Key value）而直接进行访问的数据结构。
  • 它通过把关键码值映射到一个位置来加快查找速度。映射函数叫做散列函数，存放记录的数组叫做散列表

应用实例：

有一个公司有新的员工来报道时，要求该员工的信息加入（id,性别，年龄，名字，住址，..）,当输入该员工id时，要求查找到该员工所有信息

思路分析：

HashTab s(EmpLinkedList[ ] empLinkedListArray) -> EmpLinkedList
0 -> emp ->emp->emp
…… -> emp ->emp->emp
n -> emp ->emp->emp
